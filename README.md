
# AANDILIK - Plateforme de Location de Matériel de Chantier 🏗️

## 📌 À propos du Projet
**AANDILIK** est une application web innovante visant à digitaliser le marché de la location de matériel de construction au Maroc. Elle facilite la mise en relation entre les propriétaires d'équipements (pelles, générateurs, échafaudages) et les professionnels du BTP ou particuliers.

## 🚀 Fonctionnalités Clés
- **Gestion d'Annonces :** Création et modification d'annonces avec photos et tarifs.
- **Système de Réservation :** Gestion complète du cycle de vie (En attente, Confirmée, Terminée).
- **Tableau de Bord Admin :** Supervision des utilisateurs, validation des annonces et statistiques.
- **Portefeuille Digital :** Suivi des revenus et gestion des commissions de la plateforme.

## 🛠️ Stack Technique
- **Frontend :** Next.js (React)
- **Backend :** Nest.js (Node.js framework)
- **Base de données :** MySQL
- **Authentification :** JWT (JSON Web Token)
